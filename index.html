Weather Stations:

    Point size is proportional to the number of confirmed cases. *

    Fill color is proportional to the number of deaths relative to the number of confirmed cases. Green points have no confirmed deaths.

    Border size is proportional to the number of critical cases relative to the number of confirmed cases. The critical:confirmed ratio is 5:1 for better visibility.

    * Countries with less than 20,000 confirmed cases have a non-proportional static point size to remain visible on the map.
